Metallstreifen, Rohre, Stangen usw. werden in Längen von 6000 Millimetern (6 Metern) geliefert. Wir möchten sie in Stücke unterschiedlicher Länge schneiden.

Schreiben Sie einen Code, der herausfindet, wie dies auf die effizienteste Weise, d.h. mit der geringstmöglichen Menge an verschwendetem Material, geschehen kann. Sie erhalten die gewünschten Rohrlängen (in Form einer Liste von ganzen Zahlen). Sie sollten die effizienteste Art des Schneidens finden und die Summe der verschwendeten Rohrlänge zurückgeben.

Eingabe: Eine 'BOM' (Bill Of Materials), eine Liste von Längen in Millimetern, die geschnitten werden müssen (Liste von Integer).

Ausgabe: Die Summe der verschwendeten Rohrlängen (Integer).

Beispiel: 

most_efficient_cutting([3000, 2200, 2000, 1800, 1600, 1300]) == 100
most_efficient_cutting([4000, 4000, 4000]) == 6000

Wie es verwendet wird:
Um den effizientesten Weg zur Nutzung der gegebenen Ressourcen zu finden.

Vorbedingung:
all(1 <= b <= 6000 for b in bom)
